# steg-scripts

Helpful Node.js scripts for various tasks

## Local Development

Clone the repository:

```bash
git clone https://github.com/your-username/steg-scripts.git
cd steg-scripts
```

Install dependencies:

```bash
pnpm install
```

Link the package globally to use the `ralph` command from any directory:

```bash
pnpm link --global
```

Verify it works:

```bash
ralph --help
```

To unlink:

```bash
npm unlink -g steg-scripts
```
